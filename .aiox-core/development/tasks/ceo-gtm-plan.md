# CEO GTM Plan Task

> Task: ceo-gtm-plan
> Agent: @ceo (Cadu)
> Elicit: true

## Objective

Planejar a estratégia de Go-To-Market para a AgendiX, focando em atrair os primeiros 100 clientes pagos.

## Workflow

### Phase 1: Market Analysis (Automated + Interactive)

1. **Elicit: Current State**
   ```
   QUAL O ESTÁGIO ATUAL DA AGENDIX?

   1. Pré-launch — produto em desenvolvimento, sem clientes
   2. Soft launch — poucos beta testers
   3. Early traction — alguns clientes pagos (< 10)
   4. Growing — 10-100 clientes pagos
   5. Escalando — 100+ clientes

   Digite o número:
   ```

2. **Elicit: Target Segment**
   ```
   QUAL SEGMENTO PRIORITÁRIO?

   1. Barbearias tradicionais (1-2 cadeiras)
   2. Barbearias premium (3+ cadeiras)
   3. Salões de beleza femininos
   4. Barbearias + salões mistos
   5. Outro (descrever)

   Digite o número:
   ```

### Phase 2: GTM Strategy

3. **Generate GTM plan covering:**
   - **Persona:** Ideal customer profile (barbershop owner demographics)
   - **Value Proposition:** B2B (management) + B2C (client booking)
   - **Channels:** Where to reach barbershop owners (Instagram, WhatsApp, local events, partnerships)
   - **Pricing:** Suggested pricing strategy (requires *escalate for final decision)
   - **Acquisition:** How to get first 10, then 50, then 100 customers
   - **Onboarding:** First 5-minute experience (mobile-first)
   - **Retention:** How to keep barbershop owners engaged
   - **Metrics:** CAC, LTV, churn rate targets

### Phase 3: Human Checkpoint (MANDATORY)

4. **Escalate pricing and investment decisions via *escalate**
   - NEVER set final pricing without human approval
   - NEVER commit marketing budget without human approval

### Phase 4: Post-Approval

5. **Delegate execution:**
   - Research → @analyst
   - Product features → @pm/@po
   - Implementation → @architect/@dev

## Rules

- ALWAYS focus on mobile-first (barbers use phones)
- ALWAYS consider B2B2C model (barbershop + end client)
- ALWAYS escalate pricing and budget decisions
