# Story: Ativação do Squad AIOS

## Contexto
Este projeto está sendo integrado ao framework Synkra AIOS. Esta story marca a ativação formal do Squad Orion para governar o desenvolvimento do projeto Rhian-Lepore.

## Critérios de Aceitação
- [ ] Arquivo `squad-config.yaml` criado e validado
- [ ] Estrutura de diretórios do AIOS Core auditada e íntegra
- [ ] Pelo menos 1 agente especialista (Explorer) validado para o contexto local
- [ ] Princípios da `constitution.md` reforçados via Gate de validação inicial

## Definição de Pronto
- [ ] `npm run dev` rodando sem erros de configuração do AIOS
- [ ] Checklist de auditoria arquitetural inicial concluído
- [ ] Resumo de ativação arquivado em `.aios-core/development/data/`
