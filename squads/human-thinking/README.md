# 🏗️ Human Thinking Squad

Squad com habilidades cognitivas humanas para análise, reformulação e melhoria contínua do projeto.

## Agentes

| Agente | Nome | Habilidade | Comando Principal |
|--------|------|------------|-------------------|
| 🧠 Thinker | Sophia | Pensamento crítico, questionamento | `*think-deep` |
| 🔄 Reformulator | Rex | Reformulação, simplificação | `*reformulate` |
| 🔍 Reviewer | Luna | Perspectiva do usuário, UX | `*ux-review` |
| 💡 Ideator | Spark | Geração de ideias, brainstorm | `*brainstorm` |
| 📐 Optimizer | Edge | Performance, qualidade, métricas | `*optimize` |

## Uso Rápido

```bash
# Ciclo completo (5 agentes em sequência)
*full-thinking-cycle

# Agentes individuais
*think-deep          # Análise profunda
*reformulate         # Simplificar/unificar
*ux-review           # Review de experiência
*brainstorm          # Gerar ideias
*optimize            # Otimizar performance
```

## Workflow

```
Sophia (questiona) → Luna (avalia UX) → Spark (gera ideias) → Rex (reformula) → Edge (otimiza)
```
