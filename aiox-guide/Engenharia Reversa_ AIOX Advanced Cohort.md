# Engenharia Reversa: AIOX Advanced Cohort

## 1. Estrutura do Programa (8 Semanas / 4 Fases)

### Fase 1: Migração (Semanas 1-2)
- **Setup Profissional:** Configuração do ambiente de desenvolvimento para IA.
- **Claude Code + AIOX:** Integração do projeto ativo com ferramentas de CLI (Claude Code) e o framework AIOX.
- **Objetivo:** Ter o projeto "conversando" com a IA de forma estruturada.

### Fase 2: Sistema (Semanas 3-4)
- **Design System Audit:** Inventário completo de componentes e estilos.
- **Atomic Design:** Implementação de componentes seguindo a metodologia Atomic Design (Atoms, Molecules, Organisms).
- **Objetivo:** Garantir consistência visual e técnica para que a IA possa gerar novas páginas sem erros.

### Fase 3: Orquestração (Semanas 5-7)
- **Configuração de Agentes:** Definição de papéis (Dev, Copy, Design, Data).
- **Workflows Avançados:** Criação de fluxos onde agentes colaboram (ex: Agente de Dados -> Agente de Estratégia -> Agente de Copy).
- **Deploy:** Colocar o sistema em produção com domínio próprio.

### Fase 4: Graduação (Semana 8)
- **Showcase:** Apresentação do projeto rodando.

## 2. Os Squads e Agentes (O que replicar)
- **AIOX Core:** 11 agentes de desenvolvimento (Arquiteto, Dev, QA, etc.).
- **Copy Squad:** 25 agentes baseados em copywriters lendários (DNA Mental).
- **Data Squad:** 6 especialistas em analytics e dados.
- **Design Squad:** 5 especialistas em Design System (baseados em Brand Frost).
- **Hormozi Squad:** 16 especialistas em estratégia de negócios ($100M Offers).

## 3. Sistemas Proprietários Mencionados
- **Ralfs:** Provavelmente um sistema de monitoramento ou orquestração.
- **Brand Frost:** Referência a Brad Frost (Atomic Design) para o Design System.
- **Workflow Copy:** Fluxos estruturados para geração de conteúdo.
- **Monitoramento:** Acompanhamento da performance dos agentes.
