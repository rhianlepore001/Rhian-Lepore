# Checklist: Thinking Quality Gate

## Validação de Qualidade do Ciclo de Pensamento

### Thinker (Sophia)
- [ ] Pelo menos 3 premissas ocultas identificadas
- [ ] Trade-offs documentados com prós e contras
- [ ] Perguntas sem resposta listadas para o owner
- [ ] Análise baseada em código real (não suposições)

### Reviewer (Luna)
- [ ] Todas as personas avaliadas (owner, profissional, cliente)
- [ ] Friction points com severidade atribuída
- [ ] Carga cognitiva avaliada por tela
- [ ] Sugestões concretas e implementáveis

### Ideator (Spark)
- [ ] Mínimo 10 ideias geradas na fase divergente
- [ ] Matriz impacto/esforço preenchida
- [ ] Top 3 ideias com pitch completo
- [ ] Viabilidade técnica verificada

### Reformulator (Rex)
- [ ] Before/After documentado para cada mudança
- [ ] Funcionalidade preservada (sem breaking changes)
- [ ] Padrões unificados identificados
- [ ] Impacto em arquivos mapeado

### Optimizer (Edge)
- [ ] Baseline medido antes de qualquer sugestão
- [ ] Ações priorizadas por impacto real
- [ ] Métricas de ganho estimadas
- [ ] Tech debt categorizado por severidade

### Síntese
- [ ] Top 10 ações consolidadas
- [ ] Quick wins separados de big bets
- [ ] Decisões que precisam de input do owner marcadas
- [ ] Relatório salvo em docs/thinking-cycles/
