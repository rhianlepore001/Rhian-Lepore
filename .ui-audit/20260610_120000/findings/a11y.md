### CRÍTICO-001: ui/Modal sem focus trap
- **Severidade:** CRÍTICO
- **Evidência:** `components/ui/Modal.tsx:56-75` — focus no dialogRef mas sem FocusTrap; tab escapa para conteúdo atrás
- **Problema:** Legacy `components/Modal.tsx:82` usa focus-trap-react; ui/Modal não
- **Impacto:** Usuário teclado perde contexto; violação WCAG 2.4.3
- **Fix:** Adicionar focus-trap-react ao Modal canônico
- **Esforço:** Baixo

### ALTO-002: Bottom nav itens sem aria-label
- **Severidade:** ALTO
- **Evidência:** `components/BottomMobileNav.tsx:30-38` — botões Agenda/Clientes com ícone+texto 10px; só Plus tem `aria-label`
- **Problema:** Texto 10px pode falhar contraste; ícones sem label explícito para SR quando texto cortado
- **Impacto:** Mobile a11y fraca para staff com leitor de tela
- **Fix:** `aria-label="Ir para Agenda"` em cada item; min text 12px ou sr-only label
- **Esforço:** Baixo

### ALTO-003: BrutalButton sm abaixo de 44px touch target
- **Severidade:** ALTO
- **Evidência:** `components/BrutalButton.tsx:48` — sm=`h-9` (36px); ui/Button sm=`min-h-[36px]`
- **Problema:** WCAG 2.5.8 target size — barbeiro com dedo grosso/errando toque
- **Impacto:** Miss-clicks em ações destrutivas em modais densos
- **Fix:** sm mínimo 44px em mobile via `@media (max-width: 768px)`
- **Esforço:** Baixo

### MÉDIO-004: z-index arbitrário 999
- **Severidade:** MÉDIO
- **Evidência:** `components/ui/Modal.tsx:80`, `components/Modal.tsx:70`, toast z-[200] em Dashboard
- **Problema:** Sem escala semântica (Impeccable layout rules)
- **Impacto:** Modais podem ficar atrás de banners fixed
- **Fix:** `--z-modal`, `--z-toast`, `--z-nav` em tokens
- **Esforço:** Baixo

### MÉDIO-005: prefers-reduced-motion parcial
- **Severidade:** MÉDIO
- **Evidência:** `index.html:1061` — regra existe; `animate-in` Tailwind em Layout/EmptyState não claramente cobertos
- **Problema:** Usuários com vestibular disorder sofrem fade-in em toda navegação
- **Impacto:** Desconforto físico; exclusão
- **Fix:** `@media (prefers-reduced-motion: reduce) { .animate-in { animation: none } }`
- **Esforço:** Baixo

### MÉDIO-006: Tabelas — cobertura de testes ui/Table mas uso limitado
- **Severidade:** MÉDIO
- **Evidência:** Finance/Clients usam layouts custom; `components/ui/Table.tsx` com testes
- **Problema:** Headers semânticos `<th scope>` não garantidos fora ui/Table
- **Impacto:** Dono usando VoiceOver no financeiro pode perder contexto de coluna
- **Fix:** Migrar listas densas para ui/Table
- **Esforço:** Médio

### BAIXO-007: Botão fechar banner com "x" literal
- **Severidade:** BAIXO
- **Evidência:** `pages/Dashboard.tsx:230-231` — `<span className="sr-only">Fechar</span>x`
- **Problema:** Funciona mas "x" visível sem aria-labelledby no button
- **Impacto:** Leve — sr-only presente
- **Fix:** Usar ícone X lucide + aria-label no button
- **Esforço:** Baixo
