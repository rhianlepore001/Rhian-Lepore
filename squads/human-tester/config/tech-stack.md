# Tech Stack — Human Tester Squad

## App Sob Teste
- **URL:** http://localhost:3000 (dev server)
- **Stack:** React 19 + TypeScript + Vite
- **Routing:** HashRouter (`#/` prefixed routes)
- **Database:** Supabase (PostgreSQL + RLS)
- **Auth:** Supabase Auth (session-based)
- **Temas:** Brutal (barbershop) / Beauty (salon)

## Ferramenta de Teste
- **Browser:** Playwright MCP (chromium)
- **Console:** DevTools via Playwright
- **Screenshots:** Playwright snapshot capability
- **Viewports:** 375x812 (mobile), 768x1024 (tablet), 1440x900 (desktop)
