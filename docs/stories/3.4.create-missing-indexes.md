## 📝 Dev Notes

**Índices a Criar:**
1. `idx_profiles_company_id` — Filtra profiles por tenant
2. `idx_appointments_user_status` — Query user appointments by status
3. `idx_appointments_company_date` — Range queries on dates
4. `idx_clients_company_name` — Search clients by name within company
5. `idx_transactions_company_date` — Financial reports by date

**SQL Batch:**
```sql
CREATE INDEX idx_profiles_company_id ON profiles(company_id);
CREATE INDEX idx_appointments_user_status ON appointments(user_id, status);
CREATE INDEX idx_appointments_company_date ON appointments(company_id, appointment_date);
CREATE INDEX idx_clients_company_name ON clients(company_id, name);
CREATE INDEX idx_transactions_company_date ON transactions(company_id, created_at);

ANALYZE;
```

**Performance Validation:**
```sql
-- Before/After comparison
EXPLAIN ANALYZE
SELECT * FROM appointments
WHERE company_id = 'abc-123' AND appointment_date > NOW() - INTERVAL '7 days';
```
