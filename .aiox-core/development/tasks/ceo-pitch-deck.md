# CEO Pitch Deck Task

> Task: ceo-pitch-deck
> Agent: @ceo (Cadu)
> Elicit: true

## Objective

Criar pitch deck para investidores, cobrindo problema, solução, mercado, tração, time e ask.

## Workflow

### Phase 1: Information Gathering (Interactive - elicit=true)

1. **Elicit: Pitch Context**
   ```
   PARA QUEM É ESTE PITCH DECK?

   1. Seed round (anjos, aceleradoras)
   2. Series A (VCs)
   3. Strategic partnership
   4. Pitch competition / Demo day

   Digite o número:
   ```

2. **Elicit: Key Data Points**
   ```
   QUAIS DADOS VOCÊ TEM DISPONÍVEIS?

   1. Número de clientes atuais
   2. Revenue/MRR
   3. CAC e LTV
   4. Team composition
   5. Market size (TAM/SAM/SOM)
   6. All of the above
   7. None (need to research)

   Digite o número:
   ```

### Phase 2: Generate Pitch Deck Structure

3. **Create standard pitch deck with:**
   - Slide 1: Cover (Logo + Tagline)
   - Slide 2: Problem (Barbershop management pain)
   - Slide 3: Solution (AgendiX value prop)
   - Slide 4: Product (Demo/screenshots)
   - Slide 5: Market Size (TAM/SAM/SOM for barbershop SaaS in Brazil)
   - Slide 6: Business Model (Pricing, revenue streams)
   - Slide 7: Traction (Metrics, growth)
   - Slide 8: Go-to-Market (Acquisition channels)
   - Slide 9: Team (Who we are)
   - Slide 10: Competition (Competitive landscape)
   - Slide 11: Financial Projections
   - Slide 12: The Ask (How much, for what)

### Phase 3: Human Approval (MANDATORY)

4. **Escalate via *escalate** — Investment amount and terms MUST be human-approved

5. **Delegate design** to @ux-design-expert if slides need visual work

## Output

Pitch deck structure with content for each slide, ready for design.

## Rules

- NEVER set the investment amount without human approval
- Focus on Brazilian barbershop/salon market specifically
- Keep slides concise (one idea per slide)
- Include mobile-first angle (barbers use phones)
