# US-0405: SetupCopilot — auto-start na primeira visita

**Epic:** EPIC-004 — Onboarding Wizard Refactor
**Prioridade:** P1 | **Blockers:** US-0402, US-0404
**Esforço:** 4h
**Lead:** @dev
**Objetivo:** Ao entrar no dashboard pela primeira vez após o onboarding, exibir um banner de boas-vindas que oferece o tour guiado — iniciando automaticamente se aceito, com opção de pular.

---

## 📖 História

**Como um** dono de salão/barbearia que acabou de completar o onboarding inicial,
**Eu quero** ser convidado a fazer um tour guiado pelo sistema logo que entro no dashboard,
**Para que** eu saiba exatamente o que configurar e onde clicar, sem precisar explorar sozinho.

---

## Contexto Técnico

### Lógica de detecção de "primeira visita"
```typescript
// Condições para exibir o banner de tour:
const isFirstVisit =
  onboardingProgress?.is_completed === false &&
  !onboardingProgress?.step_data?.guided_started &&
  !onboardingProgress?.step_data?.guided_dismissed_at;
```

### Banner de boas-vindas (copy)
```
"Quer um tour guiado pelo sistema?"
Subtítulo: "Vamos configurar tudo em menos de 5 minutos."
Botões: [Iniciar Tour →]  [Pular por agora]
```

### Comportamento dos botões
- **"Iniciar Tour":**
  1. Salvar `guided_started: true` em `onboarding_progress.step_data`
  2. Chamar `startGuide(firstPendingStepId)` do `GuidedModeContext`
  3. Navegar para a página do primeiro step pendente
  4. Banner desaparece

- **"Pular por agora":**
  1. Salvar `guided_dismissed_at: new Date().toISOString()` em `step_data`
  2. Banner desaparece e não reaparece em visitas futuras
  3. SetupCopilot permanece visível como lista interativa (guided mode manual disponível)

### Primeiro step pendente
```typescript
// Ordem de prioridade dos steps:
const STEP_PRIORITY: WizardStepId[] = ['services', 'team', 'hours', 'profile', 'booking', 'appointment'];
const firstPendingStep = STEP_PRIORITY.find(id => !checks[checkKeyFor(id)]);
```

- [x] **AC1:** Banner exibido apenas quando `is_completed = false` + sem `guided_started` + sem `guided_dismissed_at`
