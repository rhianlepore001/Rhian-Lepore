# CEO KPI Dashboard Task

> Task: ceo-kpi-dashboard
> Agent: @ceo (Cadu)
> Elicit: false (analysis + definition)

## Objective

Definir KPIs e métricas de negócio para monitorar a saúde e crescimento da AgendiX.

## Workflow

### Phase 1: KPI Categories

1. **Define KPIs for each category:**

   **Acquisition:**
   - Number of new barbershops signed up (weekly/monthly)
   - CAC (Customer Acquisition Cost)
   - Sign-up to paid conversion rate
   - Channel performance (Instagram, WhatsApp, referrals)

   **Activation:**
   - Time to first appointment booked
   - % of signups that complete onboarding
   - % of barbershops that send first booking link

   **Retention:**
   - Monthly churn rate
   - Weekly active barbershops
   - Average appointments per barbershop per week

   **Revenue:**
   - MRR (Monthly Recurring Revenue)
   - ARPU (Average Revenue Per User)
   - LTV (Lifetime Value)
   - LTV:CAC ratio

   **Growth:**
   - Net Promoter Score (NPS)
   - Referral rate
   - MoM growth rate

### Phase 2: Dashboard Template

2. **Generate KPI dashboard template** with:
   - Metric name
   - Target value
   - Current value (to be filled)
   - Status (Green/Yellow/Red)
   - Measurement frequency

### Phase 3: Output

3. **Present dashboard and suggest:**
   - Which metrics to track daily, weekly, monthly
   - Alert thresholds for Red status
   - Who is responsible for each metric

## Rules

- Keep it simple — no more than 10-12 primary KPIs
- Focus on actionable metrics (not vanity metrics)
- Consider what a barbershop owner can easily understand
