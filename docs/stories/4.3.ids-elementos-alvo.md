# US-0403: IDs nos elementos-alvo (contratos do wizard)

**Epic:** EPIC-004 — Onboarding Wizard Refactor
**Prioridade:** P1 | **Blocker:** US-0402
**Esforço:** 2h
**Lead:** @dev
**Objetivo:** Criar a fonte de verdade centralizada `WIZARD_TARGETS.ts` e adicionar os IDs estáveis nos elementos-alvo de cada página para que o `WizardPointer` possa encontrá-los via `document.getElementById`.

---

## 📖 História

**Como um** desenvolvedor implementando o wizard guiado,
**Eu quero** ter uma fonte de verdade centralizada que mapeia cada step do wizard para o elemento-alvo correto em cada página,
**Para que** o WizardPointer sempre aponte para o lugar certo e nenhum ID seja hardcoded espalhado pelo código.

---

## Contexto Técnico

O `WizardPointer` usa `document.getElementById(targetElementId)` para aplicar o spotlight e calcular a posição da seta. Para funcionar em qualquer página do dashboard, cada página-alvo precisa ter um `id` estável no elemento correto.

### Arquivo a criar: `constants/WIZARD_TARGETS.ts`

```typescript
export const WIZARD_TARGETS = {
  services: {
    elementId: 'btn-add-service',
    path: '/configuracoes/servicos',
    position: 'bottom' as const,
    message: 'Clique aqui para adicionar um serviço',
  },
  team: {
    elementId: 'btn-add-team-member',
    path: '/configuracoes/equipe',
    position: 'bottom' as const,
    message: 'Adicione os profissionais da sua equipe',
  },
  hours: {
    elementId: 'business-hours-section',
    path: '/configuracoes/agendamento',
    position: 'top' as const,
    message: 'Configure seus dias e horários de atendimento',
  },
  profile: {
    elementId: 'profile-logo-upload',
    path: '/configuracoes/perfil',
    position: 'right' as const,
    message: 'Adicione a logo e foto de capa do seu espaço',
  },
  booking: {
    elementId: 'toggle-public-booking',
    path: '/configuracoes/agendamento-publico',
    position: 'bottom' as const,
    message: 'Ative para receber agendamentos online',
  },
  appointment: {
    elementId: 'btn-new-appointment',
    path: '/agenda',
    position: 'bottom' as const,
    message: 'Crie seu primeiro agendamento aqui',
  },
} as const;

export type WizardStepId = keyof typeof WIZARD_TARGETS;
```

### IDs a adicionar por página

| Página | Elemento | ID a adicionar |
|--------|----------|---------------|
| `/configuracoes/servicos` | Botão "Adicionar Serviço" | `id="btn-add-service"` |
| `/configuracoes/equipe` | Botão "Adicionar Membro" | `id="btn-add-team-member"` |
| `/configuracoes/agendamento` | Seção de horários | `id="business-hours-section"` |
| `/configuracoes/perfil` | Campo de upload da logo | `id="profile-logo-upload"` |
| `/configuracoes/agendamento-publico` | Toggle agendamento online | `id="toggle-public-booking"` |
| `/agenda` | Botão "Novo Agendamento" | `id="btn-new-appointment"` |
