## Sobre

Este conjunto de regras foi criado para orientar IDEs Agênticas (Cursor, Antigravity, Claude Code, Windsurf, Cline) na geração de código seguro, performático e manutenível.

Cada regra define:
- **MOTIVO**: Por que a regra existe
- **GATILHO**: Quando ela deve ser aplicada
- **EXEMPLOS**: Código errado vs. correto

## Como Usar

1. Coloque todos os arquivos `.md` no diretório `.agent/rules/` do seu projeto
2. A IDE agêntica irá automaticamente aplicar essas regras durante a geração de código
3. Revise periodicamente para garantir conformidade

## Contribuindo

Sinta-se livre para adaptar essas regras às necessidades do seu projeto. Esta é uma base sólida que pode ser expandida conforme sua stack e requisitos específicos.
