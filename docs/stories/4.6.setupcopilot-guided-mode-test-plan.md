# US-0406 Test Plan: SetupCopilot Guided Mode Per Step

## Overview
Complete test coverage for guided mode activation, visual indicators, and behavior across all 6 SetupCopilot steps.

---

## Test Environment Setup

**Prerequisites:**
1. User logged in with complete onboarding (dashboard accessible)
2. SetupCopilot card visible on dashboard
3. Browser console open for error checking
4. At least one SetupCopilot step NOT completed

**Note:** Testing guided mode requires navigating away from dashboard and back (portal rendering test).

---

## Test Cases

### TC-0406-001: Tour Banner First Visit Detection
**Objective:** Verify tour banner shows on first dashboard visit
**Steps:**
1. Clear `onboarding_progress.step_data.guided_started` flag (set to false/null)
2. Clear `onboarding_progress.step_data.guided_dismissed_at` (set to null)
3. Navigate to dashboard fresh
4. **Expected:** Tour banner appears with animation (fade-in, slide-from-top)

**Pass Criteria:**
- Banner visible ✓
- Contains text "Quer um tour guiado pelo sistema?" ✓
- Two buttons visible: "Iniciar Tour" + "Pular por agora" ✓
- X close button visible (top-right) ✓

---

### TC-0406-002: Tour Banner Click "Iniciar Tour"
**Objective:** Start guided mode on first pending step
**Steps:**
1. Click "Iniciar Tour" button
2. **Expected:** Immediate actions:
   - Banner animates out (fade-out)
   - Dashboard SetupCopilot updates state
   - Browser navigates to first pending step's page (e.g., /configuracoes/servicos)

**Pass Criteria:**
- URL changes to correct page ✓
- No console errors ✓
- Tour banner gone ✓

---

### TC-0406-003: Tour Banner Click "Pular por agora"
**Objective:** Dismiss tour and save dismissed flag
**Steps:**
1. Click "Pular por agora" button (or X)
2. **Expected:**
   - Banner animates out
   - Database updated: `guided_dismissed_at` = current ISO timestamp
   - Banner doesn't reappear on same session

**Pass Criteria:**
- Banner gone ✓
- Refresh page → banner still gone ✓
- Verify DB: `SELECT step_data->>'guided_dismissed_at' FROM onboarding_progress WHERE company_id = {user_id}` returns timestamp ✓

---

### TC-0406-004: Step Click Activates Guided Mode
**Objective:** Verify clicking a pending step activates guided mode
**Steps:**
1. From dashboard SetupCopilot, click on a pending step (e.g., "Cadastrar serviços")
2. **Expected:**
   - Navigate to step's page
   - "Parar tutorial" button appears in SetupCopilot header (if revisiting dashboard)
   - `onboarding_progress.step_data.last_guided_step` = clicked step ID

**Pass Criteria:**
- URL correct ✓
- No errors ✓
- DB saved: `SELECT step_data->>'last_guided_step' FROM onboarding_progress` returns step ID ✓

---

- No flickering ✓
