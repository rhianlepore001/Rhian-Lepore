## Como Criar uma Nova Story

1. Chame `@sm` (River) com uma descrição do que precisa ser feito
2. O agente criará o arquivo `[YYYY-MM-DD]-[id]-[slug].md` neste diretório
3. Revise os critérios de aceitação
4. Chame `@dev` para implementar

## Convenção de Nomes

- `[YYYY-MM-DD]` = Data de criação
- `[id]` = Número sequencial (US-001, US-002, etc.)
- `[slug]` = Descrição curta em kebab-case

**Exemplo:** `2026-02-22-US-001-adicionar-memoria-semantica.md`
