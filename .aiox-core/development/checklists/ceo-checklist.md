# CEO Checklist — AgendiX

> Use este checklist para validar decisões do CEO antes de prosseguir.

## Business Decision Checklist

Antes de tomar qualquer decisão de negócio:

- [ ] A decisão foi escalada para o humano (fundador) via `*escalate`?
- [ ] Existem dados/métricas que suportam a decisão?
- [ ] O impacto no cliente B2B (barbearia) foi considerado?
- [ ] O impacto no cliente B2C (agendador) foi considerado?
- [ ] O aspecto mobile-first foi considerado?
- [ ] O impacto financeiro (custo/receita) foi estimado?
- [ ] Riscos foram identificados e documentados?

## Hire Plan Checklist

Antes de aprovar um plano de contratação:

- [ ] O gap de time foi identificado via `*team-review`?
- [ ] O perfil da vaga foi definido via `*role-define`?
- [ ] O custo total (salário + impostos + benefícios) foi calculado?
- [ ] A vaga foi mapeada para um agente AIOX (para delegação)?
- [ ] A vaga foi aprovada pelo humano via `*escalate`?
- [ ] O cronograma de contratação foi definido?
- [ ] O modelo de contratação (CLT/PJ/Freelance) foi definido?

## Business Idea Checklist

Antes de prosseguir com uma ideia de negócio:

- [ ] A ideia foi gerada via `*business-idea`?
- [ ] O score ICE foi calculado?
- [ ] A ideia foi escalada para o humano via `*escalate`?
- [ ] O impacto na receita foi estimado?
- [ ] O esforço de implementação foi estimado?
- [ ] A ideia foi delegada ao agente correto (PM, CTO, Analyst)?

## Delegation Checklist

Antes de delegar uma tarefa:

- [ ] A tarefa é técnica? → Delegar para @architect (CTO)
- [ ] A tarefa é de produto? → Delegar para @pm
- [ ] A tarefa é de pesquisa? → Delegar para @analyst
- [ ] A tarefa é de implementação? → Delegar para @dev
- [ ] A tarefa envolve decisão de negócio? → ESCALAR via `*escalate`
- [ ] O contexto da AgendiX foi passado para o agente?

## Escalation Checklist

Quando escalar uma decisão para o humano:

- [ ] A categoria da decisão foi identificada (Pricing/Investment/Pivot/Partnership/Hiring/Financial)?
- [ ] Pelo menos 2 opções foram apresentadas com prós e contras?
- [ ] A recomendação do CEO foi incluída com fundamentação?
- [ ] O impacto financeiro foi estimado?
- [ ] O prazo para a decisão foi definido?
- [ ] A decisão foi registrada em MEMORY.md?
