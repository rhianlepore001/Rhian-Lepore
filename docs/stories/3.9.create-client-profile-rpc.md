## 📝 Dev Notes

**Problema Atual (3 queries sequenciais):**
```typescript
// ClientCRM.tsx - SLOW (3 round trips)
const client = await supabase
  .from('clients')
  .select('*')
  .eq('id', clientId)
  .single();

const appointments = await supabase
  .from('appointments')
  .select('*')
  .eq('client_id', clientId)
  .limit(10); // Problem: N+1 LIMIT

const transactions = await supabase
  .from('transactions')
  .select('*')
  .eq('client_id', clientId)
  .limit(5);
```

**Solução (1 RPC call):**
```sql
CREATE OR REPLACE FUNCTION get_client_profile(p_client_id UUID)
RETURNS TABLE (
  client_id UUID,
  client_name TEXT,
  client_email TEXT,
  recent_appointments JSON,
  recent_transactions JSON
) AS $$
WITH client_data AS (
  SELECT id, name, email
  FROM clients
  WHERE id = p_client_id
    AND company_id = (SELECT company_id FROM auth.jwt())
),
appointments_data AS (
  SELECT json_agg(row_to_json(a)) as appointments
  FROM (
    SELECT * FROM appointments
    WHERE client_id = p_client_id
    ORDER BY appointment_date DESC
    LIMIT 10
  ) a
),
transactions_data AS (
  SELECT json_agg(row_to_json(t)) as transactions
  FROM (
    SELECT * FROM transactions
    WHERE client_id = p_client_id
    ORDER BY created_at DESC
    LIMIT 5
  ) t
)
SELECT c.id, c.name, c.email, a.appointments, t.transactions
FROM client_data c
CROSS JOIN appointments_data a
CROSS JOIN transactions_data t;
$$ LANGUAGE SQL;
```

**React Integration:**
```typescript
// ClientCRM.tsx - FAST (1 round trip)
const { data } = await supabase
  .rpc('get_client_profile', { p_client_id: clientId })
  .single();

const client = {
  id: data.client_id,
  name: data.client_name,
  appointments: JSON.parse(data.recent_appointments),
  transactions: JSON.parse(data.recent_transactions),
};
```
