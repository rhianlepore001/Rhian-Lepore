# US-0406: SetupCopilot — guided mode por step

**Epic:** EPIC-004 — Onboarding Wizard Refactor
**Prioridade:** P1 | **Blockers:** US-0403, US-0404, US-0405
**Esforço:** 8h
**Lead:** @dev
**Objetivo:** Transformar cada step do SetupCopilot em um gatilho de guided mode — ao clicar, navega para a página correta e o WizardPointer aparece apontando para o elemento-alvo com spotlight.

---

## 📖 História

**Como um** usuário configurando meu espaço no AgenX,
**Eu quero** que ao clicar em um step do painel de configuração, o sistema me leve diretamente para onde preciso agir e uma seta me mostre exatamente o que clicar,
**Para que** eu nunca fique perdido sem saber o que fazer a seguir.

---

## Contexto Técnico

### Refatoração do click handler no SetupCopilot

Hoje:
```typescript
onClick={() => navigate(step.path)}
```

Proposto:
```typescript
onClick={() => {
  startGuide(step.id as WizardStepId);  // ativa Context + WizardPointer
  navigate(WIZARD_TARGETS[step.id].path); // navega para a página
}}
```

### Comportamento completo por step

| Step | Página destino | Elemento alvo | Posição |
|------|---------------|---------------|---------|
| services | /configuracoes/servicos | `btn-add-service` | bottom |
| team | /configuracoes/equipe | `btn-add-team-member` | bottom |
| hours | /configuracoes/agendamento | `business-hours-section` | top |
| profile | /configuracoes/perfil | `profile-logo-upload` | right |
| booking | /configuracoes/agendamento-publico | `toggle-public-booking` | bottom |
| appointment | /agenda | `btn-new-appointment` | bottom |

### Step "team" — opcional
- Badge visual `"Opcional"` ao lado do label
- Step pode ser ignorado sem bloquear o wizard
- Se clicado, guided mode funciona normalmente

### Controle do tour ativo
Enquanto guided mode está ativo, exibir no dashboard (canto superior direito do SetupCopilot):
```
[◼ Parar tutorial]
```
- Clique: `endGuide()` + salva estado no sessionStorage
- Não navega o usuário de volta — apenas desativa o pointer

### Indicador visual do step ativo
- Step sendo guiado no momento: highlight com borda `accent-gold` / `beauty-neon`
- Demais steps: opacidade reduzida enquanto guided mode ativo em outro step
