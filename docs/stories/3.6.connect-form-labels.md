# US-0306: Conectar labels de formulário via `htmlFor`/`id`

**Epic ID:** EPIC-003-S1 (Sprint 1)
**Prioridade:** P1 (Quick Win)
**Esforço:** 2 horas
**Lead:** Uma (@ux-design-expert)
**Objetivo:** Todos os inputs de formulário estão semanticamente conectados aos seus labels

---

## 📖 História

**Como um** desenvolvedor de acessibilidade,
**Eu quero** que cada `<input>` tenha um `<label htmlFor={id}>` conectado,
**Para que** leitores de tela anunciem o label corretamente e usuários cliquem no label para focar no input.

---

## ✅ Critérios de Aceitação

- [x] **AC1:** Todos os inputs têm `id` único
- [x] **AC2:** Todos os labels têm `htmlFor={inputId}` correspondente
- [ ] **AC3:** Testado: clicar no label foca o input
- [ ] **AC4:** Testado com leitor de tela: label é pronunciado

## 📝 Dev Notes

**Padrão de Implementação:**
```tsx
// Before
<label>Email</label>
<input type="email" />

// After
<label htmlFor="email-input">Email</label>
<input id="email-input" type="email" />
```
